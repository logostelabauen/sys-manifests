
# ops-handbook

A minimalist knowledge base and documentation hub focusing on infrastructure design, network administration, and systems engineering. This repository serves as a personal reference for reproducible environments and secure configurations.

## 📂 Structure

```text
.
├── applications/
│   └── lms-deployment.md       # Application provisioning and service setups
├── networking/
│   ├── administration.md       # Routing protocols, access control, and admin guides
│   └── topology-config.md      # Server node configurations and network topologies
└── systems-engineering/
    ├── archlinux-ops.md        # Core system maintenance and environment configs
    └── storage-encryption/     # Cryptographic disk layouts and volume management
        ├── luks-over-lvm.md
        └── lvm-over-luks.md
```

## 🛠️ Core Focus & Technologies

- **Systems & OS:** Linux system administration, custom kernel adjustments, and automated environment provisioning (Arch Linux).
- **Security & Storage:** Cryptographic data protection, LUKS block encryption, and LVM (Logical Volume Management) architectures.
- **Infrastructures:** Decoupled service deployments, network routing topologies, and localized server configurations.

## 📝 Usage

These documents are written in standard Markdown and meant for architectural reference. To navigate through the configurations, simply explore the directories above or use a local markdown viewer.
